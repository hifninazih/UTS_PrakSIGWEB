Esai Kegiatan Praktikum

Kegiatan Praktikum Sebagai mahasiswa Universitas Gadjah Mada jurusan Teknik Geodesi angkatan 2022, saya memiliki kesempatan untuk mengikuti mata kuliah Praktikum SIG Web yang sangat menarik. Mata kuliah ini membahas tentang pengembangan web map yang sangat relevan dengan bidang studi saya. Pada minggu ke-2, saya dan teman-teman saya diajarkan tentang cara membuat form validasi yang efektif. Kegiatan ini sangat bermanfaat karena saya dapat memahami bagaimana membuat form yang dapat memvalidasi input pengguna dengan benar. Pada minggu ke-3, kami diajarkan tentang cara membuat halaman web dengan menggunakan Bootstrap. Framework ini sangat membantu saya dalam membuat halaman web yang responsif dan mudah digunakan. Pada minggu ke-5, kami diajarkan tentang cara membuat tampilan peta sederhana dengan menggunakan Leafletjs. Library ini sangat membantu saya dalam membuat peta yang interaktif dan dapat diintegrasikan dengan data lainnya.

Selain itu, pada minggu ke-5, kami juga diajarkan tentang cara membuat halaman web sederhana dengan menggunakan HTML, CSS, dan Javascript. Kegiatan ini sangat bermanfaat karena saya dapat memahami bagaimana membuat halaman web yang dapat menampilkan data dengan efektif. Dengan demikian, saya dapat memahami bagaimana membuat aplikasi web yang dapat menampilkan data dengan efektif dan interaktif. Saya sangat berharap bahwa pengetahuan dan keterampilan yang saya dapatkan dari mata kuliah ini dapat saya aplikasikan dalam proyek-proyek yang akan saya kerjakan di masa depan.

Jika penasaran dengan hal yang dipelajari di Teknik Geodesi UGN, kamu dapat berkunjung ke halaman resmi https://geodesi.ugm.ac.id/ .
